# GeneGuard-Smart-AI-Based-Genetic-Risk-Tracker
GeneGuard Smart AI-Based Genetic Risk Tracker is a modern full-stack web application designed to help users understand and manage potential genetic health risks. It provides an interactive platform where users can input family health history and receive AI-assisted insights, visual risk analysis, and personalized health tracking.

The system includes features like:

Genetic risk analysis dashboard
Family history visualization
Health passport generation
AI chatbot assistance for health-related queries
Interactive charts and reports for better understanding of risk patterns

Built with a scalable architecture using modern web technologies, GeneGuard aims to make genetic health awareness more accessible, intuitive, and data-driven.
